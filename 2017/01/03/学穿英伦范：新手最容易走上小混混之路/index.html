举几个例子，Roman Style 的形成和英国人有着密切的联系。在18世纪末期，英国的年轻人中流行泛欧旅行（Grand Tour）。在这个过程中，萨维尔街的剪裁被带到了罗马，今日罗马西装还是留下很多萨街的痕迹的。比如说较厚的胸衬和肩垫。另外，罗马地处意大利北部，冬天也是可以很 chilly的，罗马的西装显然不能像西西里西装那样里面做出无衬或者半衬的。

现在让我们谈一谈是什么造成了这些男装的风尚，以后大家可以拿这个 check list 去分析其他国家的着装（美国啊，霓虹啊）。

首先是自然原因，为什么英国人用克重大的世佳宝？冬天伦敦好冷的。厚实一些的西装可以让英国人春秋穿三件套出门而不用 overcoat。为什么意大利南部西装把衬里做的尽可能的薄？夏天地中海地区真的好热。

然后是历史原因，萨维尔街男装的肩垫事实上是军装留下来的传统。号称萨维尔街第一家的 Gieves&Hawkes 以前就是做军装的。萨街男装中常见的票兜（Ticket Pocket），是两个世纪以前设计用来装剧院票的。

还有文化原因，意大利人似乎是非常注意线条的。在他们的西装中，gorge line 被拉的更高来塑造躯干的线条；在单排扣的西装中，枪驳头的要比英国少的多（宰你丫那种给美国人跪舔的先不说），因为不用条纹不做双排扣西装的话，枪驳头很容易给人一种突兀的感觉，丧失了平驳头 notches 很高的那种流畅的线条感。我在上文说的意大利人用色的大胆鲜艳，应该也算是文化上的原因吧。

一口气写了这么多，现在看似乎有些离题了呢。毕竟所谓风尚，是风度在前时尚在后，历史上真正引领潮流的，爱德华七世也好，布鲁梅尔也好，温莎公爵也好，他们都是胜在风度而不是单单在衣着上。

穿衣这种事情，还是看个人气质吧…

我知道有人肯定问我，你写了这么多衣服牌子，我怎么一个也不认识啊？说实话，我不是什么富二代，一个还没上大学的孩子，平时也不过穿一穿Giorgio Armani，Zenga这些大众品牌。去年在港岛金钟的 WW Chan家做了一套 Bespoke，的的确确感觉到了穿成衣和穿定制的差别。一套 Brioni 也许是85分，Chan或许是89分，我想 Henry&Poole 可能就是99分了吧，这其中的差距，我想只有试过的人才有体会吧……在我心目中，RL、Dunhill、Giorgio Armani 之流，不过是高级打工仔和农民企业家的穿着，曾经的Gieves&Hawkes、Huntsman 和 Kiton 才真正是绅士的穿着，才是男装的最高水平。

有的同学问我怎么不在一个更大层面上谈一谈英伦范，说实话，我很想，可是我没有这个能力。什么马丁靴啊什么英伦摇滚范啊我真的是不懂，生活中很少接触过那样打扮的人。在知乎我不想不懂装懂，即便是谈我稍微了解的萨维尔街剪裁，我也在文章第一句就讲我是个外行。毕竟服装行业的水真是太深了，我可不想贻笑大方。

也有同学问我为什么不写一写便宜大众一点的牌子，像是 Burberry，Armani 这些。我个人认为英国男装之剪裁之风尚还是发源于萨维尔街。Burberry 现在产品线分的很细，像 Burberry Brit 旗下的西装已经部分地丧失了英国西装的特点了。所以尽管Burberry的 Trench Coat 是那么的流行，但是我不会写Burberry，因为不够 authentic，因为文不对题。

最后我要向广大英伦摇滚迷道个歉……我真的不知道那张我说是小混混的图片是一个 band，那张图片只是我随手在 Google image 上搜的。不过无论如何你们也不能否认十年以前英国混混真的是穿成那个样子的。
